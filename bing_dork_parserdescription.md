# 🌑⚔️ Shadow Realm Bing Dork Parser ⚔️🌑
**Automated Bing Dork URL Harvester**
⚡ Fast • Optimized • Stable • Premium

---

## 🚀 Overview
Shadow Realm Bing Dork Parser is a multi-threaded tool that executes search dorks on Bing to harvest target URLs. It reads dorks from a file, queries Bing for each one, extracts unique URLs from results, and supports proxy rotation for high-speed, anonymous harvesting.

---

## 🎮 Features

### 🔍 Harvesting Features
- Dork file input support
- Bing search query automation
- Unique URL extraction per dork
- Proxy rotation support
- Configurable thread count

### ⚙️ System Features
- ⚡ ThreadPoolExecutor multi-threaded
- 📊 tqdm live progress display
- 📡 Proxy support for anonymity
- 📁 All URLs saved to output file
- 🔄 Rate limiting and retry logic

---

## 📦 Installation

```bash
1. Download the ZIP file
2. Extract the archive
3. Install dependencies:
   pip install -r "bing_dork_parser requirements.txt"
4. Run the script:
   python bing_dork_parser.py -d dorks.txt -o output.txt
```
